Project vision

Viewing the attendance counts per each sport for a given school name and year, how many spectators were there for each sport?

Initial results

    FROM uni_sport_attendance('James Madison', 2018)
    ORDER BY attendance DESC, sport;

Project vision

Obtain the number of players for each sport for a university and year.

Initial results

    SELECT sport, count(sport) as c
    FROM uni_players('James Madison', 2018)
    GROUP BY sport
    ORDER BY count(sport) DESC;
